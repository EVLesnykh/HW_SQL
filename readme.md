# SQL Basics Practice

## Описание
Этот репозиторий содержит учебные задания и практические упражнения по SQL.  
Цель — закрепление базовых навыков работы с базами данных, включая создание запросов, работу с таблицами, группировку данных и использование агрегатных функций.

Проект включает задания из домашних работ (HW1–HW6) и практику по лекциям 1–6.

---

## Используемые навыки и инструменты
- SQL (SELECT, INSERT, UPDATE, DELETE)  
- Работа с несколькими таблицами  
- JOIN, GROUP BY, агрегатные функции  
- Фильтрация и сортировка данных  
- Практика построения запросов на основе учебных данных  

---

## Цель репозитория
- Закрепление теоретических знаний SQL на практике  
- Подготовка к реальным задачам по работе с базами данных
